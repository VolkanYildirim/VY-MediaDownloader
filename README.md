### ⚙️ Security & Privacy Philosophy (Secure by Design)
As a core engineering standard, this project is strictly built on **Privacy-First** principles:
- **Zero Telemetry:** The application does not collect, log, or transmit any user data, download history, or network activity to third parties.
- **Zero Leak:** There are no hardcoded credentials, hidden API keys, or tracking scripts within the source code.
- **Local Processing:** All parsing, bypassing, and downloading operations are executed entirely on the local host machine.

### 🚀 Key Features
- **Security Algorithm Bypass:** Effectively navigates recent platform security protocols and VPN-related blocks (Introduced in V2).
- **Format Flexibility:** Supports automated extraction of high-quality Video (MP4) and Audio-only formats.
- **Modern UI:** Built with `CustomTkinter` for a responsive, user-friendly, and fatigue-free dark mode interface.
